    Suppose we wanted to calculate how much it costs to heat a house on a particular day of the year. The cost to heat a house will depend on the average daily temperature, and the average daily temperature depends on the particular day of the year. Notice how we have just defined two relationships: The temperature depends on the day, and the cost depends on the temperature. Using descriptive variables, we can notate these two functions.
    ðŸ”—
    The first function, \(f(T)\text{,}\) gives the cost \(C\) of heating a house when the average daily temperature is \(T\) degrees Celsius, and the second, \(g(d)\text{,}\) gives the average daily temperature on day \(d\) of the year in some city. If we wanted to determine the cost of heating the house on the 5th day of the year, we could do this by linking our two functions together, an idea called composition of functions. Using the function \(g(d)\text{,}\) we could evaluate \(g(5)\) to determine the average daily temperature on the 5th day of the year. We could then use that temperature as the input to the \(f(T)\) function to find the cost to heat the house on the 5th day of the year: \(f(g(5))\text{.}\)
    ðŸ”—

    Definition 1.4.1. Composition of Functions.

    When the output of one function is used as the input of another, we call the entire operation a composition of functions. We write \(f(g(x))\) and read this as "\(f\) of \(g\) of \(x\)" or "\(f\) composed with \(g\) at \(x\)".
    ðŸ”—
    An alternate notation for composition uses the composition operator: \(\circ\)
    ðŸ”—
    \(f\circ g(x)\) is read "\(f\) of \(g\) of \(x\)" or "\(f\) composed with \(g\) at \(x\)", just like \(f(g(x))\text{.}\) We could refer to the name of this new function as \(f\circ g\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.4.2.

    Suppose \(f(s)\) gives the number of calories burned doing \(s\) sit-ups, and \(g(t)\) gives the number of sit-ups a person can do in \(t\) minutes. Interpret \(f(g(3))\text{.}\)
    ðŸ”—

    Solution.

    When we are asked to interpret, we are being asked to explain the meaning of the expression in words. The inside expression in the composition is \(g(3)\text{.}\) Since the input to the \(g\) function is time, the 3 is representing 3 minutes, and \(g(3)\) is the number of sit-ups that can be done in 3 minutes. Taking this output and using it as the input to the \(f(s)\) function will gives us the calories that can be burned by the number of sit-ups that can be done in 3 minutes.
    ðŸ”—
    ðŸ”—
    ðŸ”—
    Note that it is no context involved in the scenario, it is not important that the same variable be used for the output of the inside function and the input to the outside function. However, it is essential that if we are given context, then the units on the output of the inside function match the units on the input to the outside function if the units are specified.
    ðŸ”—

    Example 1.4.3.

    Suppose \(f(x)\) gives miles that can be driven in \(x\) hours, and \(g(y)\) gives the gallons of gas used in driving \(y\) miles. Which of these expressions is meaningful: \(f(g(y))\) or \(g(f(x))\text{?}\)
    ðŸ”—

    Solution.

    The expression \(g(y)\) takes miles as the input and outputs a number of gallons. The function \(f(x)\) is expecting a number of hours as the input; trying to give it a number of gallons as input does not make sense. Remember the units must match, and number of gallons does not match number of hours, so the expression \(f(g(y))\) is meaningless.
    ðŸ”—
    The expression \(f(x)\) takes hours as input and outputs a number of miles driven. The function \(g(y)\) is expecting a number of miles as the input, so giving the output of the \(f(x)\) function (miles driven) as an input value for \(g(y)\text{,}\) where gallons of gas depend on miles driven, does make sense. The expression \(g(f(x))\) makes sense and will give the number of gallons of gas used, driving a certain number of miles, \(f(x)\text{,}\) in \(x\) hours.

    Subsection 1.4.2 Composition of Functions using Tables and Graphs

    When working with functions given as tables and graphs, we can look up values for the functions using a provided table or graph, as discussed in section 1.2. We start evaluation from the provided input, and first evaluate the inside function. We can then use the output of the inside function as the input to the outside function. To remember this, always work from the inside out, as if we are following the order of operations.
    ðŸ”—

    Example 1.4.5.

    Using the tables below, evaluate \(f(g(3))\) and \(g(f(4))\text{.}\)
    ðŸ”—
    \(x\) \(f(x)\)
    1 6
    2 8
    3 3
    4 1
    \(x\) \(g(x)\)
    1 3
    2 5
    3 2
    4 7

    Solution.

    To evaluate \(f(g(3))\text{,}\) we start from the inside with the value 3. We then evaluate the inside expression \(g(3)\) using the table that defines the function \(g\text{:}\) \(g(3)=2\text{.}\) We can then use that result as the input to the \(f\) function, so \(g(3)\) is replaced by the equivalent value 2 and we can evaluate \(f(2)\text{.}\) Then using the table that defines the function \(f\text{,}\) we find that \(f(2)=8\text{.}\) Thus, \(f(g(3))=f(2)=8\text{.}\)
    ðŸ”—
    To evaluate \(g(f(4))\text{,}\) we first evaluate the inside expression \(f(4)\) using the first table: \(f(4)=1\text{.}\) Then using the table for \(g\text{,}\) we can evaluate: \(g(f(4))=g(1)=3\text{.}\)

    ðŸ”—
    ðŸ”—

    Subsection 1.4.3 Composition using Formulas

    When evaluating a composition of functions where we have either created or been given formulas, the concept of working from the inside out remains the same. First, we evaluate the inside function using the input value provided, then use the resulting output as the input to the outside function.
    ðŸ”—

    Example 1.4.11.

    Given \(f(t)=t^{2}-t\) and \(h(x)=3x+2\text{,}\) evaluate \(f(h(1))\text{.}\)
    ðŸ”—

    Solution.

    Since the inside evaluation is \(h(1)\text{,}\) we start by evaluating the \(h(x)\) function at 1:
    ðŸ”—
    \(h(1)=3(1)+2=5\)
    ðŸ”—
    Then \(f(h(1))=f(5)\text{,}\) so we evaluate the \(f(t)\) function at an input of 5:
    ðŸ”—
    \(f(h(1))=f(5)=5^{2}-5=20\text{.}\)
    ðŸ”—
    ðŸ”—
    ðŸ”—

    While we can compose the functions as above for each individual input value, sometimes it would be really helpful to find a single formula which will calculate the result of a composition \(f(g(x))\text{.}\) To do this, we will extend our idea of function evaluation. Recall that when we evaluate a function like \(f(t)=t^{2}-t\text{,}\) we put whatever value is inside the parentheses after the function name into the formula wherever we see the input variable.
    ðŸ”—

    Example 1.4.13.

    Given \(f(t)=t^{2}-t\text{,}\) evaluate \(f(3)\) and \(f(-2)\text{.}\)
    ðŸ”—

    Solution.

    \(f(3)=3^{2}-3\)
    ðŸ”—
    \(f(-2)=(-2)^{2}-(-2)\)
    ðŸ”—
    We could simplify the results above if we wanted to
    ðŸ”—
    \(f(3)=3^{2}-3=9-3=6\)
    ðŸ”—
    \(f(-2)=(-2)^{2}-(-2)=4+2=6\)
    ðŸ”—
    ðŸ”—
    ðŸ”—
    We are not limited, however, to using a numerical value as the input to the function. We can put anything into the function: a value, a different variable, or even an algebraic expression, provided we use the input expression everywhere we see the input variable.
    ðŸ”—

    Example 1.4.14.

    Using the function from the previous example, evaluate \(f(a)\text{.}\)
    ðŸ”—

    Solution.

    This means that the input value for \(t\) is some unknown quantity \(a\text{.}\) As before, we evaluate by replacing the input variable \(t\) with the input quantity, in this case \(a\text{.}\)
    ðŸ”—
    \(f(a)=a^{2}-a\)
    ðŸ”—
    ðŸ”—
    ðŸ”—
    The same idea can then be applied to expressions more complicated than a single letter.
    ðŸ”—

    Example 1.4.15.

    Using the same \(f(t)\) function from above, evaluate \(f(x+2)\text{.}\)
    ðŸ”—

    Solution.

    Everywhere in the formula for \(f\) where there was a \(t\text{,}\) we would replace it with the input \(x+2\text{.}\) Since in the original formula the input \(t\) was squared in the first term, the entire input \(x+2\) needs to be squared when we substitute, so we need to use grouping parentheses. To avoid problems, it is advisable to always use parentheses around inputs.
    ðŸ”—
    \(f(x+2)=(x+2)^{2}-(x+2)\)
    ðŸ”—
    We could simplify this expression further to \(f(x+2)=x^{2}+3x+2\) if we wanted to:
    ðŸ”—
    \(f(x+2)=(x+2)(x+2)-(x+2)\)
    ðŸ”—
    Use the "FOIL" technique (first, outside, inside, last)
    ðŸ”—
    \(f(x+2)=x^{2}+2x+2x+4-(x+2)\)
    ðŸ”—
    distribute the negative sign
    ðŸ”—
    \(f(x+2)=x^{2}+2x+2x+4-x-2\)
    ðŸ”—
    combine like terms
    ðŸ”—
    \(f(x+2)=x^{2}+3x+2\text{.}\)
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Using the same function, evaluate \(f(x^{3})\text{.}\)
    ðŸ”—

    Solution.

    We replace the original input \(t\) in the formula with the new input expression, \(x^{3}\text{.}\)
    ðŸ”—
    \(f(x^{3})=(x^{3})^{2}-(x^{3})=x^{6}-x^{3}\text{.}\)

    This now allows us to find an expression for a composition of functions. If we want to find a formula for \(f(g(x))\text{,}\) we can start by writing out the formula for \(g(x)\text{.}\) We can then evaluate the function \(f(x)\) at that expression, as in the examples above.

    Example 1.4.18.

    Let \(f(x)=x^{2}\) and \(g(x)=\dfrac{1}{x}-2x\text{,}\) find \(f(g(x))\) and \(g(f(x))\text{.}\)
    ðŸ”—

    Solution.

    To find \(f(g(x))\text{,}\) we start by evaluating the inside, writing out the formula for \(g(x)\text{.}\)
    ðŸ”—
    \(g(x)=\dfrac{1}{x}-2x\)
    ðŸ”—
    We then use the expression \(\dfrac{1}{x}-2x\) as input for the function \(f\text{.}\)
    ðŸ”—
    \(f(g(x))=f(\dfrac{1}{x}-2x)\)
    ðŸ”—
    We then evaluate the function \(f(x)\) using the formula for \(g(x)\) as the input.
    ðŸ”—
    Since \(f(x)=x^{2}\text{,}\) \(f(\dfrac{1}{x}-2x)=(\dfrac{1}{x}-2x)^{2}\)
    ðŸ”—
    This gives us the formula for the composition: \(f(g(x))=(\dfrac{1}{x}-2x)^{2}\text{.}\)
    ðŸ”—
    Likewise, to find \(g(f(x))\text{,}\) we evaluate the inside, writing out the formula for \(f(x)\text{:}\)
    ðŸ”—
    \(g(f(x))=g(x^{2} )\)
    ðŸ”—
    Now we evaluate the function \(g(x)\) using \(x^{2}\) as the input.
    ðŸ”—
    \(g(f(x))=\dfrac{1}{x^2}-2x^{2}\)
    ðŸ”—
    ðŸ”—

    A city manager determines that the tax revenue, \(R\text{,}\) in millions of dollars collected on a population of \(p\) thousand people is given by the formula \(f(p)=0.03p+\sqrt{p}\text{,}\) and that the cityâ€™s population, in thousands, is predicted to follow the formula \(g(t)=60+2t+0.3t^{2}\text{,}\) where \(t\) is measured in years after 2010.     Subsection 1.4.4 Domain of Compositions

    When we think about the domain of a composition \(h(x)=f(g(x))\text{,}\) we must consider both the domain of the inner function and the domain of the composition itself. While it is tempting to only look at the resulting composite function, if the inner function were undefined at a value of \(x\text{,}\) the composition would not be possible.
    ðŸ”—

    Example 1.4.21.

    Let \(f(x)=\dfrac{1}{x^2-1}\) and \(g(x)=\sqrt{x-2}\text{.}\) Find the domain of \(f(g(x))\text{.}\)
    ðŸ”—

    Solution.

    Since we want to avoid the square root of negative numbers, the domain of \(g(x)\) is the set of values where \(x-2\geq 0\text{.}\) The domain is \(x\geq 2\text{.}\)
    ðŸ”—
    The composition is \(f(g(x))=\dfrac{1}{(\sqrt{x-2})^2-1}=\dfrac{1}{(x-2)-1}=\dfrac{1}{x-3}\text{.}\) The composition is undefined when \(x = 3\text{,}\) so that value must also be excluded from the domain. Notice that the composition doesnâ€™t involve a square root, but we still have to consider the domain limitation from the inside function. Combining the two restrictions, the domain is all values of \(x\) greater than or equal to 2, except \(x = 3\text{.}\)
    ðŸ”—
    In inequalities, the domain is: \(2\leq x<3\) or \(x>3\text{.}\) In interval notation, the domain is: \([2,3)\cup(3,\infty)\text{.}\)
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Subsection 1.4.5 Decomposing Functions

    In some cases, it is desirable to decompose a function - to write it as a composition of two simpler functions.
    ðŸ”—

    Example 1.4.23.

    Write \(f(x)=3+\sqrt{5-x^{2}}\) as the composition of two functions.
    ðŸ”—

    Solution.

    We are looking for two functions, \(g\) and \(h\text{,}\) so \(f(x)=g(h(x))\text{.}\) To do this, we look for a function inside a function in the formula for \(f(x)\text{.}\) As one possibility, we might notice that \(5-x^{2}\) is the inside of the square root. We could then decompose the function as:
    ðŸ”—
    \(h(x)=5-x^{2}\)
    ðŸ”—
    \(g(x)=3+\sqrt{x}\)
    ðŸ”—
    We can check our answer by recomposing the functions: \(g(h(x))=g(5-x^{2})=3+\sqrt{5-x^{2}}\)
    ðŸ”—
    Note that this is not the only solution to the problem. Another non-trivial decomposition would be \(h(x)=x^{2}\) and \(g(x)=3+\sqrt{5-x}\text{.}\)
